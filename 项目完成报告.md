## 🚀 如何使用

### 快速启动（3步）

```bash
# 1. 安装依赖
npm install

# 2. 创建数据库并配置
createdb voice_badcase

# 3. 启动服务
./start.sh
```

### 访问地址

- **前端**：http://localhost:5173

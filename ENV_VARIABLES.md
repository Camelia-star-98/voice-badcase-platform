# 环境变量模板 - 复制到 Vercel

## 使用说明

在 Vercel 部署时，将以下环境变量添加到项目配置中：

### Supabase 配置
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

### 钉钉机器人配置
- `DINGTALK_APP_KEY`
- `DINGTALK_APP_SECRET`
- `DINGTALK_AGENT_ID`
- `DINGTALK_CORP_ID`
- `DINGTALK_TOKEN`
- `DINGTALK_AES_KEY`

## 获取这些值

从你本地的 `.env` 文件中复制对应的值到 Vercel 的环境变量设置中。

⚠️ **重要**：确保变量名完全一致（区分大小写）

# 数据可视化使用说明

## 📊 功能概览

数据可视化页面提供了6种不同类型的图表，帮助您全面了解语音Badcase的数据情况。

## 🎯 图表说明

### 1. Badcase趋势分析图（折线图）

**位置**：左上方大图

**功能**：
- 展示一周内Badcase的新增、已解决和待处理数量趋势
- 支持时间范围筛选
- 可以对比不同状态的数据变化

**使用方法**：
- 鼠标悬停查看具体数值
- 点击图例可以隐藏/显示对应的数据线
- 使用顶部的时间范围选择器调整展示时间段

**解读提示**：
- 新增曲线上升 → Badcase数量增加，需要关注
- 已解决曲线高于新增 → 处理效率良好
- 待处理曲线持续上升 → 可能需要增加处理资源

---

### 2. Badcase分类分布图（饼图）

**位置**：右上方

**功能**：
- 展示各类型Badcase的占比情况
- 直观了解哪些类型的问题最多

**使用方法**：
- 鼠标悬停查看具体数量和百分比
- 点击图例可以隐藏/显示对应的分类

**解读提示**：
- 某个分类占比过高 → 该类型问题需要重点优化
- 分布均衡 → 问题较为分散，需要多方面优化

**主要分类**：
- 🎤 语音识别错误
- 🔊 语音合成问题
- 🎵 音质问题
- ⚡ 语速问题
- 📝 其他

---

### 3. 严重程度统计图（柱状图）

**位置**：中间左侧

**功能**：
- 按严重程度统计Badcase数量
- 快速识别高优先级问题

**使用方法**：
- 查看每个柱子顶部的数字
- 对比不同严重程度的数量

**解读提示**：
- 🔴 高严重度数量多 → 需要紧急处理
- 🟡 中严重度占比大 → 日常优化重点
- 🟢 低严重度较多 → 可以安排后续处理

**严重程度说明**：
- **高**：严重影响用户体验，需立即处理
- **中**：有一定影响，需尽快处理
- **低**：影响较小，可延后处理

---

### 4. 处理状态分布图（环形图）

**位置**：中间右侧

**功能**：
- 展示当前所有Badcase的处理状态
- 监控整体处理进度

**使用方法**：
- 查看各状态的具体数量
- 通过占比了解整体进度

**解读提示**：
- ✅ 已解决占比高 → 处理效率好
- ⏳ 处理中较多 → 正在积极处理
- ⚠️ 待处理占比高 → 需要加快处理速度

**状态说明**：
- **已解决**：问题已修复并验证通过
- **处理中**：正在分析和修复中
- **待处理**：已记录但尚未开始处理

---

### 5. Badcase时段热力图

**位置**：底部左侧

**功能**：
- 展示Badcase在一周不同时段的分布规律
- 帮助发现高发时段

**使用方法**：
- 颜色越深表示该时段Badcase数量越多
- 横轴为24小时，纵轴为星期
- 鼠标悬停查看具体时段和数量

**解读提示**：
- 发现高发时段 → 在该时段加强监控
- 识别低发时段 → 可能是系统使用率低
- 规律性出现 → 可能与特定业务场景相关

**实际应用**：
- 制定测试计划时参考高发时段
- 安排值班和监控资源
- 分析业务场景与Badcase的关联

---

### 6. 各类别指标对比图（雷达图）

**位置**：底部右侧

**功能**：
- 多维度对比本周和上周的指标表现
- 全面评估系统质量

**使用方法**：
- 查看两条线的差异
- 每个顶点代表一个指标
- 面积越大表示整体表现越好

**指标说明**：
- **识别准确度**：语音识别的准确率
- **响应速度**：系统响应时间
- **音质评分**：合成语音的音质质量
- **用户满意度**：用户反馈评分
- **系统稳定性**：系统运行稳定程度

**解读提示**：
- 🔵 本周线高于 🟢 上周线 → 该指标有提升
- 某个指标明显偏低 → 需要重点优化
- 整体面积增大 → 系统质量在提升

## 🔍 筛选功能使用

### 时间范围选择
1. 点击"时间范围"日期选择器
2. 选择开始日期和结束日期
3. 点击"确定"应用筛选
4. 所有图表会根据选定时间范围更新

### 分类筛选
1. 点击"分类"下拉框
2. 选择要查看的分类（或选择"全部"）
3. 图表会只显示该分类的数据

### 严重程度筛选
1. 点击"严重程度"下拉框
2. 选择要查看的严重程度（高/中/低）
3. 图表数据会相应更新

### 组合筛选
- 可以同时使用多个筛选条件
- 例如：查看"上周"的"语音识别错误"类型中"高严重度"的Badcase

## 🛠️ 功能按钮

### 🔄 刷新按钮
- **功能**：重新获取最新数据
- **使用场景**：数据更新后需要立即查看
- **位置**：筛选区域右侧

### 💾 导出数据按钮
- **功能**：导出当前筛选条件下的数据
- **导出格式**：CSV/Excel
- **使用场景**：需要进行深度分析或报告

## 💡 使用技巧

### 1. 日常监控
- 每天查看趋势分析图，了解Badcase变化
- 关注待处理数量，及时调整资源

### 2. 问题定位
- 使用热力图找出高发时段
- 结合分类分布图定位主要问题类型

### 3. 效果评估
- 使用雷达图对比改进前后的效果
- 观察已解决/新增比例的变化

### 4. 报告制作
1. 设置合适的时间范围
2. 应用必要的筛选条件
3. 点击导出按钮获取数据
4. 使用图表截图作为报告素材

### 5. 团队协作
- 定期查看状态分布图，了解处理进度
- 根据严重程度统计分配任务优先级

## ❓ 常见问题

### Q1: 图表数据不显示？
**A**: 检查筛选条件是否过于严格，尝试重置筛选器。

### Q2: 如何保存图表？
**A**: 右键点击图表，选择"保存图片"即可。

### Q3: 数据什么时候更新？
**A**: 点击刷新按钮会获取最新数据，也可以设置自动刷新。

### Q4: 可以自定义图表样式吗？
**A**: 目前使用默认主题，后续版本会支持自定义。

### Q5: 导出的数据在哪里？
**A**: 默认保存在浏览器的下载文件夹中。
